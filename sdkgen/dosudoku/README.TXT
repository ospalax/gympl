SDKgen je program ke generovani
a reseni SUDOKU.

Slouzi pouze k osobnim,
nekomercnim ucelum s cilem
pobavit. Program je freeware,
ale nesmi byt modifikovan a ani
s nim nesmi byt zachazeno jinak,
nez tak jak bylo zamysleno
autorem (viz vyse).

Hlavolamy ziskane sebemensim
prispenim programu nesmi byt
publikovany, pouze po domluve s
autorem.

Autorem SDKgen je Petr Ospaly.
